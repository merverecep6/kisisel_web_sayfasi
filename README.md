# Kişisel Web Sayfası (Web CBS - Ödev 1)

Bu proje, **Web CBS – Ödev 1 Kişisel Web Sayfası** gereksinimlerine göre hazırlanmıştır:
- En az 3 farklı başlık ve HTML dosyası (Hakkımda / Hobilerim / İletişim)
- Hobilerim sayfasında **Leaflet** haritası
- CSS animasyon
- GitHub Pages ile yayınlanabilir

## Klasör Yapısı
- `index.html` Ana sayfa
- `hakkimda.html` Hakkımda
- `hobilerim.html` Harita + tiyatro listesi
- `iletisim.html` İletişim
- `pages/` Her oyun için ayrı HTML sayfası
- `css/style.css` Stil ve animasyonlar
- `js/theatreData.js` Tiyatro verileri (mekan/koordinat)
- `js/map.js` Leaflet yardımcı fonksiyonları
- `assets/` Fotoğraf ve CV (PDF)

## Çalıştırma
Dosyalar statik olduğu için doğrudan tarayıcıda açabilirsiniz:
- `index.html` dosyasını çift tıklayın.

## GitHub Pages Yayınlama
1. Bu klasörü bir GitHub reposuna yükleyin.
2. GitHub’da: **Settings → Pages**
3. Source: `Deploy from a branch`, Branch: `main`, Folder: `/ (root)`
4. Kaydedince birkaç dakika içinde link oluşur.
